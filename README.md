# 🇰🇿 哈萨克西里尔字母学习 / Kazakh Cyrillic Alphabet Learning

一个为哈萨克语学习者设计的西里尔字母入门工具，包含发音音频、字母表和常用词汇。

## 📦 项目内容

- **index.html** — 主页，含有三个学习入口
- **哈萨克西里尔字母入门.html** — 用哈萨克语学习的字母表页面
- **哈萨克西里尔字母入门-哈语版.html** — 哈语原版版
- **用中文学哈语.html** — 用中文解释的哈语学习页面
- **sounds/** — 所有字母和单词的发音音频文件（WAV格式）

## 💻 离线使用

下载整个项目到电脑上，双击 `index.html` 即可使用，不需要网络。

## 🎯 学习内容

- 所有哈萨克语西里尔字母及其发音
- 每个字母配有标准发音音频
- 常用单词和示范词汇
- 中哈双语说明
