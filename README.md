## 📦 使用方式

### 方式一：直接打开
双击 `index.html` 即可使用，完全离线，无需网络。

### 方式二：安装为应用（推荐）
1. 用 Chrome/Edge 打开 `index.html`
2. 地址栏出现安装图标 → 点击安装
3. 以后在桌面直接打开，像 App 一样用

## 🎯 学习路径

1. **首页** → 选择 "汉语→西里尔字母" 
2. 先记住 **9个特殊字母**（哈萨克独有的）
3. 再学 **元音**（注意元音和谐规则）
4. 然后 **辅音**
5. 做 **测验** 和 **闪卡** 巩固
6. 看 **学习统计** 追踪进度
